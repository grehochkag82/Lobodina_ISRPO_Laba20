### Что изучили в ходе лабораторной работы

В ходе выполнения лабораторной работы №20 я изучил следующие темы:

**1. DOM**
- Что такое DOM,его древовидная структура
- Как JavaScript взаимодействует с DOM

**2. Поиск элементов в DOM**
- `getElementById()` — поиск элемента по уникальному id
- `querySelector()` — универсальный поиск по CSS-селектору
- Различия и особенности методов поиска

**3. Изменение содержимого и стилей**
- Свойство `textContent` для изменения текста
- Свойство `style` для изменения CSS-стилей
- Динамическое изменение внешнего вида элементов

**4. Работа с событиями**
- `addEventListener()` — назначение обработчиков событий
- Событие `click` — реакция на клик мыши
- Событие `input` — реакция на ввод текста

**5. Работа с формами и вводом данных**
- Получение значений из полей ввода (`.value`)
- `event.preventDefault()` — отмена стандартного поведения
- Обработка данных формы на стороне клиента

**6. Динамическое создание элементов**
- `createElement()` — создание новых HTML-элементов
- `appendChild()` — добавление элементов в DOM
- Создание текстовых узлов
- Динамическое добавление кнопок и обработчиков

**7. Делегирование событий**
- Проблема обработки динамически созданных элементов
- Решение через делегирование событий
- Преимущества: один обработчик на родителя
**8. Сравнение DOM с C# WinForms/WPF**
- Аналогии между DOM и Controls
- Сравнение подходов к обработке событий
